# installation

Mecabのインストール

```Bash
$ sudo apt install mecab

$ sudo apt install libmecab-dev

$ sudo apt install mecab-ipadic-utf8

$ sudo apt install swig

$ sudo cp /etc/mecabrc /usr/local/etc/
```

```Bash
$ mecab --version
mecab of 0.996

```

`mecab-python`の導入

```
$ pip install mecab-python3
```
